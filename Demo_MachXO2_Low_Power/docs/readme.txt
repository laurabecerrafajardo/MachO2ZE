This lower power control demo is based on MachXO2 Pico board.

Folder structure:
docs: contains readme.txt and UG58.pdf.
Pico_I2C_SPI_master: contains the design for I2C/SPI master based on MachXO2 Pico board,
                     including Diamond project and MICO8 project.
power_control: contains the Diamond project for user and configure mode demos.
